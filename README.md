## 🚀 Flagship & Featured Projects

### 🍔 **KhaoPio – Restaurant POS System** (Flagship Monorepo)
*Welcome to KhaoPio, a modern, high-performance, and feature-rich Point of Sale (POS) system designed for restaurant workflows.*
* **Frontend**: Built with **Next.js 16 (App Router)**, **React 19**, **Tailwind CSS v4**, and **Zustand** state management. Features an intuitive ordering cart, staff permission-based sidebars, live kitchen display, and cashier billing system.
* **Backend**: Powered by **Node.js**, **Express**, **TypeScript**, and **Prisma ORM** connected to **PostgreSQL**. Features JWT security, staff PIN authentication, mail invitations, order/KOT workflows, and menu manipulation APIs.
* **Links**: [GitHub Repository](https://github.com/rishav161/KhaoPio) | [Live Demo](https://khao-pio.vercel.app/login)

### ⚡ **Real-Time Collaboration Dashboard**
*A modern full-stack collaboration platform enabling real-time communications and sync.*
* **Tech Stack**: Next.js, TypeScript, Prisma, PostgreSQL, Socket.IO, Clerk, Zustand, Tailwind CSS.
* **Features**: Real-time 1-to-1 and group chats, active typing indicators, live online status sync, Clerk multi-factor authentication, and custom team invite links.
* **Links**: [GitHub Repository](https://github.com/rishav161/realtime-collab-dashboard)

### 🌍 **Tripease – Travel & Tourism Booking Portal**
*A production-ready travel bookings application handling transactions and itinerary setups.*
* **Tech Stack**: React, Node.js, Express, PostgreSQL, Payment Gateway Integration.
* **Features**: Role-based access for tourists/agents, secure payment checkout pipelines, dynamic booking management, and location explorer APIs.
* **Links**: [GitHub Repository](https://github.com/sudipt-31/Tripease)

### 🎮 **Pokédex Application**
*A web app displaying live Pokemon statistics with quick search capabilities.*
* **Tech Stack**: React, Next.js, TypeScript, PokeAPI.
* **Features**: Responsive multi-device catalog, fast client-side query filters, and comprehensive data visualizers for combat stats.
* **Links**: [GitHub Repository](https://github.com/rishav161/pokedex)
